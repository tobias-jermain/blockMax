# Code of Conduct

Please be respectful when interacting with this repository.

## Expectations

- Be respectful and constructive.
- Keep discussions focused on the project.
- Report bugs and suggest features in good faith.
- Provide clear and useful information when opening issues or pull requests.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Spam or malicious content.
- Intentionally disruptive behavior.
- Offensive or abusive language.

Contributors who repeatedly violate these guidelines may have their comments, issues, or pull requests removed.

Thank you for helping keep blockMax professional and productive.
